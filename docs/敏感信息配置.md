## 🛡️ 安全注意事项

1. **定期更换密码**：建议每3-6个月更换数据库密码
2. **JWT密钥轮换**：生产环境应定期更换JWT密钥
3. **SSL证书续期**：Let's Encrypt证书每90天自动续期
4. **访问控制**：限制SSH访问IP范围
5. **备份加密**：所有备份文件应加密存储

## 📝 密码强度要求

- **数据库密码**：至少12位，包含大小写字母、数字、特殊字符
- **JWT密钥**：256位随机字符串
- **用户密码**：至少8位，包含大小写字母、数字、特殊字符
